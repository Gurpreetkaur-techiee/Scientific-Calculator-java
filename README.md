# Scientific-Calculator-java
A beginner -friendly Scientific Calculator developed in java using Scanner, Switch Case , Loops, and Math Class
## features
-Addition
-Subtraction
-Multiplication
-Division
-Power
-Square root
-Sin
-Cos
-Tan
-Logarithm
-Factorial
## Technologies Used
-Java
-Scanner Class
-Switch Case
-Loops
-Math class
